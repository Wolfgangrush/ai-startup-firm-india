## ✨ ਇਹ ਕੀ ਹੈ, ਇੱਕ paragraph ਵਿੱਚ

ਇੱਕ ਮੁਫ਼ਤ open-source shell ਜੋ Claude Code ਨੂੰ ਤੁਹਾਡੇ startup ਦਾ legal co-pilot ਬਣਾਉਂਦਾ ਹੈ। ਇਹ founders' agreements, iSAFE notes, SHAs, employment contracts, ਅਤੇ POSH policies draft ਕਰਦਾ ਹੈ। ਤੁਹਾਡੇ ਸਵਾਲਾਂ ਨੂੰ 10 startup intents ਰਾਹੀਂ 8 specialist agents ਕੋਲ route ਕਰਦਾ ਹੈ, ਜੋ 5 Wolfgang Rush drafting plugins ਚਲਾਉਂਦੇ ਹਨ। ਇਹ ਤੁਹਾਡੀ ਕੰਪਨੀ ਨੂੰ ਯਾਦ ਰੱਖਦਾ ਹੈ — founders, cap table, open matters, compliance deadlines — session ਤੋਂ session। ਤੁਹਾਡੀ ਆਪਣੀ machine ਉੱਤੇ Claude Code ਦੇ ਅੰਦਰ ਚੱਲਦਾ ਹੈ। Cloud upload ਨਹੀਂ। Subscription ਨਹੀਂ। ਜੋ document ਬਣਦਾ ਹੈ, ਉਹ ਤੁਹਾਡਾ।

## 💛 ਇੱਕ Indian founder ਨੂੰ ਕਿਉਂ ਪਰਵਾਹ ਕਰਨੀ ਚਾਹੀਦੀ

ਵੱਡੀਆਂ law firms ਇੱਕ standard SHA ਲਈ ₹50,000–₹2,00,000 ਲੈਂਦੀਆਂ ਹਨ। Bengaluru ਜਾਂ Mumbai ਵਿੱਚ ਇੱਕ ਚੰਗਾ startup lawyer ਘੰਟੇ ਦੇ ₹5,000–₹15,000 bill ਕਰਦਾ ਹੈ। ਤੁਸੀਂ pre-revenue ਹੋ। Bootstrapping ਕਰ ਰਹੇ ਹੋ। ਤੁਹਾਨੂੰ ਕੰਮ ਕਰਨ ਵਾਲੇ documents ਚਾਹੀਦੇ — runway ਖਤਮ ਕਰਨ ਵਾਲਾ relationship ਨਹੀਂ।

- **Drafting ਵਿੱਚ ਲੱਖਾਂ ਬਚਾਓ।** Founders' Agreement, iSAFE, SHA, Employment Contracts — templated, jurisdiction-aware, ਤੁਹਾਡੇ counsel ਦੀ ਸਮੀਖਿਆ ਲਈ ਤਿਆਰ।
- **Default ਰੂਪ ਵਿੱਚ compliant ਰਹੋ।** Companies Act 2013, FEMA NDI Rules, DPDP Act 2023, POSH Act 2013 — compliance agent ਦੱਸਦਾ ਹੈ ਤੁਹਾਡੇ stage 'ਤੇ ਕੀ ਲਾਗੂ ਹੁੰਦਾ।
- **ਕਦੇ context ਨਾ ਗੁਆਓ।** Firm ਤੁਹਾਡੀ cap table, incorporation state, open drafts — sessions ਦੇ ਵਿਚਕਾਰ, ਹਫ਼ਤਿਆਂ ਬਾਅਦ ਵੀ ਯਾਦ ਰੱਖਦੀ।
- **ਤੇਜ਼ੀ ਨਾਲ ਅੱਗੇ ਵਧੋ।** Plain English (ਜਾਂ Hindi, Marathi, Tamil...) ਵਿੱਚ ਦੱਸੋ ਕੀ ਚਾਹੀਦਾ — firm ਸਹੀ specialist ਕੋਲ route ਕਰਦੀ।
- **ਸਭ ਕੁਝ ਤੁਹਾਡਾ।** MIT licensed. ਤੁਹਾਡੇ documents. ਤੁਹਾਡਾ data. ਤੁਹਾਡਾ laptop.

## 🧠 ਅੰਦਰ ਕੀ ਹੈ — 8 specialists ਤੁਹਾਡੇ terminal ਵਿੱਚ

8 startup lawyers ਵਾਂਗ ਸੋਚੋ, ਹਰੇਕ ਦੀ ਆਪਣੀ lane:

1. **Receptionist (ਦਿਮਾਗ)** — ਤੁਸੀਂ ਜੋ ਕਹਿੰਦੇ ਹੋ ਸੁਣਦਾ, 10 startup flows ਵਿੱਚ intent classify ਕਰਦਾ, ਸਹੀ specialist ਕੋਲ route ਕਰਦਾ।
2. **Matter Manager** — ਤੁਹਾਡੀਆਂ entity details, founders, cap table, open incorporations, pending drafts ਰੱਖਦਾ। Sessions ਵਿਚਕਾਰ stateful।
3. **Companies Act Compliance Officer** — Companies Act 2013: MoA, AoA, DIN, DSC, Section 10A, statutory registers, annual filings. ਤੁਹਾਡੇ stage 'ਤੇ ਕੀ ਲਾਗੂ ਹੈ, ਜਾਣਦਾ।
4. **FEMA Rules Specialist** — foreign investment: NDI Rules, FC-GPR, FMV computation, pricing guidelines। RBI approval ਦੀ ਲੋੜ ਪੈਣ 'ਤੇ flag ਕਰਦਾ।
5. **DPDP Overlay** — Data Protection Act 2023: consent notices, data fiduciary obligations, breach reporting thresholds।
6. **Stamp Duty Router** — 4 ਰਾਜਾਂ (MH, KA, DL, TS) ਵਿੱਚ stamp duty compute ਕਰਦਾ, ਦੱਸਦਾ ਕਿੱਥੇ ਤੇ ਕਿੰਨਾ ਭਰਨਾ।
7. **POSH Compliance Officer** — workplace harassment prevention: policy drafting, IC constitution, annual filings। Non-compliant ਹੋਣ ਤੋਂ ਪਹਿਲਾਂ ਫੜਦਾ।
8. **Funding & News Watch** — ਕਿਸਨੇ funding raise ਕੀਤੀ, VC deal flow, Startup India scheme updates, YC batches, accelerator deadlines.

ਤੁਸੀਂ ਸਿਰਫ਼ Receptionist ਨਾਲ ਗੱਲ ਕਰਦੇ ਹੋ। Receptionist ਬਾਕੀਆਂ ਨਾਲ ਗੱਲ ਕਰਦਾ।

## 🚀 ਤੁਹਾਡੇ ਪਹਿਲੇ 30 ਮਿੰਟ (quick start)

### Prerequisites
- Claude Code installed
- Node.js 18+ ਅਤੇ Python 3.10+
- Git

### Step 1 — Firm clone (2 ਮਿੰਟ)

```bash
git clone https://github.com/Wolfgangrush/ai-startup-firm-india.git ~/Desktop/AIO/ai-startup-firm/india/
```

### Step 2 — 5 Wolfgang Rush drafting plugins install (5 ਮਿੰਟ)

```bash
git clone https://github.com/Wolfgangrush/wolfgang-rush-submissions.git ~/Desktop/AIO/wolfgang-rush-submissions/
```

5 startup drafting plugins `~/Desktop/AIO/wolfgang-rush-submissions/IN-startup/` ਵਿੱਚ:

| # | Plugin | ਕੀ draft ਕਰਦਾ |
|---|---|---|
| 1 | `startup-india-founders-agreement-drafting` | Founders' Agreement, Co-founder Exit + Vesting |
| 2 | `startup-india-incorporation-drafting` | MoA + AoA, DSC + DIN, Section 10A declaration |
| 3 | `startup-india-isafe-drafting` | iSAFE INR, iSAFE Foreign (FEMA overlay) |
| 4 | `startup-india-sha-drafting` | SHA Standard, SHA Investment (FEMA overlay) |
| 5 | `startup-india-employment-drafting` | Employment Contract, Offer Letter, POSH + HR Policies |

### Step 3 — Session start (30 ਸਕਿੰਟ)

```bash
cd ~/Desktop/AIO/ai-startup-firm/india/
claude
```

Claude Code prompt 'ਤੇ:

```
/wake
```

Saved state load ਕਰਦਾ। ਪਹਿਲੀ ਵਾਰ ਹੈ ਤਾਂ fresh firm initialize ਕਰਦਾ। Startup dashboard ਦੇਖੋਗੇ — company snapshot, open matters, compliance deadlines, funding news।

### Step 4 — ਪਹਿਲੀ ਗੱਲਬਾਤ

ਦੱਸੋ ਕੀ ਚਾਹੀਦਾ:

```
> Acme Tech ਲਈ founders' agreement draft ਕਰੋ — 3 co-founders, 4-year vesting, 1-year cliff
> Karnataka ਵਿੱਚ registered ਕੰਪਨੀ ਦੇ SHA 'ਤੇ stamp duty ਕਿੰਨੀ?
> ₹50L angel round ਲਈ iSAFE note ਚਾਹੀਦਾ
> Check ਕਰੋ ਕਿ 10th employee hire ਕਰਨ 'ਤੇ POSH compliance ਚਾਹੀਦੀ?
> ਇਸ ਹਫ਼ਤੇ Indian SaaS ਵਿੱਚ ਕਿਸਨੇ funding raise ਕੀਤੀ?
```

Receptionist intent classify ਕਰਕੇ ਸਹੀ drafting plugin ਜਾਂ specialist agent ਕੋਲ route ਕਰਦਾ।

ਹਰ plugin 6-agent drafting pipeline (Reader → Format → Drafter → Verifier → Refiner → Overseer) ਚਲਾ ਕੇ `.docx` file ਤੁਹਾਡੀ working directory ਵਿੱਚ ਦਿੰਦਾ।

### Step 5 — ਹਰ session ਦੇ ਅੰਤ 'ਤੇ `/retrospective` ਚਲਾਓ

```
/retrospective
```

ਤੁਹਾਡੇ company facts, cap table state, open matters, decisions local palace ਵਿੱਚ save ਕਰਦਾ। ਅਗਲੀ session ਵਿੱਚ `/wake` ਸਭ load ਕਰਦਾ। Firm ਸਿੱਖਦੀ।

## 🛡️ Privacy ਦੀ ਇਮਾਨਦਾਰ ਗੱਲ

**Tool ਆਪ:** ਤੁਹਾਡਾ company data, cap table, drafts, notes — ਸਭ laptop 'ਤੇ। Cloud 'ਤੇ ਕਦੇ upload ਨਹੀਂ। ਸਭ `~/.ai-startup-firm-india/palace/` ਵਿੱਚ। Source code MIT licensed — ਹਰ ਲਾਈਨ ਪੜ੍ਹੋ, audit ਕਰੋ, modify ਕਰੋ। Drafting plugins `.docx` files local disk 'ਤੇ ਬਣਾਉਂਦੇ।

**AI model (Claude Code):** Claude Code Anthropic ਦੀ API ਨਾਲ connect ਹੁੰਦਾ। ਤੁਹਾਡੀਆਂ queries ਉਹਨਾਂ ਦੇ servers 'ਤੇ ਜਾਂਦੀਆਂ। Anthropic ਦੇ enterprise privacy defaults strong — prompts ਅਤੇ outputs default ਰੂਪ ਵਿੱਚ model training ਲਈ use ਨਹੀਂ ਹੁੰਦੇ। ਹੋਰ: https://docs.claude.codes

**Compliance ਜ਼ਿੰਮੇਵਾਰੀ:** Compliance agents ਦੱਸਦੇ ਤੁਹਾਡੇ stage 'ਤੇ ਕੀ ਲਾਗੂ ਹੈ। ਉਹ ਤੁਹਾਡੇ company secretary, CA, ਜਾਂ legal counsel ਦੇ substitute ਨਹੀਂ। RoC filing, stamp duty ਭਰਨਾ, POSH IC constitution, ਅਤੇ ਸਾਰੀਆਂ statutory obligations ਦੀ ਜ਼ਿੰਮੇਵਾਰੀ ਤੁਹਾਡੀ।
