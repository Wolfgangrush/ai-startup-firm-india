## ✨ இது என்ன, ஒரு பத்தியில்

ஒரு இலவச open-source shell — Claude Code-ஐ உங்கள் startup-இன் legal co-pilot ஆக மாற்றுகிறது. இது founders' agreements, iSAFE notes, SHAs, employment contracts, மற்றும் POSH policies draft செய்கிறது. உங்கள் கேள்விகளை 10 startup intents மூலம் 8 specialist agents-இடம் route செய்கிறது, அவை 5 Wolfgang Rush drafting plugins-ஐ இயக்குகின்றன. இது உங்கள் நிறுவனத்தை நினைவில் வைக்கிறது — founders, cap table, open matters, compliance deadlines — session முதல் session வரை. உங்கள் சொந்த machine-இல் Claude Code-இன் உள்ளே இயங்குகிறது. Cloud upload இல்லை. Subscription இல்லை. உருவாகும் document உங்களுடையது.

## 💛 ஒரு Indian founder ஏன் கவலைப்பட வேண்டும்

பெரிய law firms ஒரு standard SHA-க்கு ₹50,000–₹2,00,000 வசூலிக்கின்றன. Bengaluru அல்லது Mumbai-இல் ஒரு நல்ல startup lawyer மணிக்கு ₹5,000–₹15,000 bill செய்கிறார். நீங்கள் pre-revenue. Bootstrapping செய்கிறீர்கள். வேலை செய்யும் documents தேவை — runway-ஐ முடிக்கும் relationship அல்ல.

- **Drafting-இல் லட்சங்கள் சேமியுங்கள்.** Founders' Agreement, iSAFE, SHA, Employment Contracts — templated, jurisdiction-aware, உங்கள் counsel-இன் மதிப்பாய்வுக்குத் தயார்.
- **Default-ஆக compliant-ஆக இருங்கள்.** Companies Act 2013, FEMA NDI Rules, DPDP Act 2023, POSH Act 2013 — compliance agent உங்கள் stage-க்கு என்ன பொருந்தும் என்று கூறுகிறது.
- **ஒருபோதும் context-ஐ இழக்காதீர்கள்.** Firm உங்கள் cap table, incorporation state, open drafts — sessions-க்கு இடையேயும், வாரங்களுக்குப் பின்னரும் நினைவில் வைக்கிறது.
- **வேகமாக நகர்த்துங்கள்.** Plain English (அல்லது Hindi, Marathi, Tamil...)-இல் என்ன வேண்டும் என்று சொல்லுங்கள் — firm சரியான specialist-இடம் route செய்கிறது.
- **அனைத்தும் உங்களுடையது.** MIT licensed. உங்கள் documents. உங்கள் data. உங்கள் laptop.

## 🧠 உள்ளே என்ன — 8 specialists உங்கள் terminal-இல்

8 startup lawyers போல நினைத்துக் கொள்ளுங்கள், ஒவ்வொருவருக்கும் சொந்த lane:

1. **Receptionist (மூளை)** — நீங்கள் சொல்வதைக் கேட்கிறது, 10 startup flows-இல் intent classify செய்கிறது, சரியான specialist-இடம் route செய்கிறது.
2. **Matter Manager** — உங்கள் entity details, founders, cap table, open incorporations, pending drafts வைத்திருக்கிறது. Sessions-க்கு இடையே stateful.
3. **Companies Act Compliance Officer** — Companies Act 2013: MoA, AoA, DIN, DSC, Section 10A, statutory registers, annual filings. உங்கள் stage-க்கு என்ன பொருந்தும் என்று தெரியும்.
4. **FEMA Rules Specialist** — foreign investment: NDI Rules, FC-GPR, FMV computation, pricing guidelines. RBI approval தேவைப்படும்போது flag செய்கிறது.
5. **DPDP Overlay** — Data Protection Act 2023: consent notices, data fiduciary obligations, breach reporting thresholds.
6. **Stamp Duty Router** — 4 மாநிலங்களில் (MH, KA, DL, TS) stamp duty compute செய்கிறது, எங்கே எவ்வளவு செலுத்த வேண்டும் என்று கூறுகிறது.
7. **POSH Compliance Officer** — workplace harassment prevention: policy drafting, IC constitution, annual filings. Non-compliant ஆவதற்கு முன் பிடிக்கிறது.
8. **Funding & News Watch** — யார் funding raised செய்தார்கள், VC deal flow, Startup India scheme updates, YC batches, accelerator deadlines.

நீங்கள் Receptionist-இடம் மட்டுமே பேசுகிறீர்கள். Receptionist மற்றவர்களிடம் பேசுகிறது.

## 🚀 உங்கள் முதல் 30 நிமிடங்கள் (quick start)

### Prerequisites
- Claude Code installed
- Node.js 18+ மற்றும் Python 3.10+
- Git

### Step 1 — Firm clone (2 நிமிடங்கள்)

```bash
git clone https://github.com/Wolfgangrush/ai-startup-firm-india.git ~/Desktop/AIO/ai-startup-firm/india/
```

### Step 2 — 5 Wolfgang Rush drafting plugins install (5 நிமிடங்கள்)

```bash
git clone https://github.com/Wolfgangrush/wolfgang-rush-submissions.git ~/Desktop/AIO/wolfgang-rush-submissions/
```

5 startup drafting plugins `~/Desktop/AIO/wolfgang-rush-submissions/IN-startup/`-இல்:

| # | Plugin | எதை draft செய்கிறது |
|---|---|---|
| 1 | `startup-india-founders-agreement-drafting` | Founders' Agreement, Co-founder Exit + Vesting |
| 2 | `startup-india-incorporation-drafting` | MoA + AoA, DSC + DIN, Section 10A declaration |
| 3 | `startup-india-isafe-drafting` | iSAFE INR, iSAFE Foreign (FEMA overlay) |
| 4 | `startup-india-sha-drafting` | SHA Standard, SHA Investment (FEMA overlay) |
| 5 | `startup-india-employment-drafting` | Employment Contract, Offer Letter, POSH + HR Policies |

### Step 3 — Session start (30 வினாடிகள்)

```bash
cd ~/Desktop/AIO/ai-startup-firm/india/
claude
```

Claude Code prompt-இல்:

```
/wake
```

Saved state-ஐ load செய்கிறது. முதல் முறை என்றால் fresh firm initialize செய்கிறது. Startup dashboard-ஐக் காண்பீர்கள் — company snapshot, open matters, compliance deadlines, funding news.

### Step 4 — முதல் உரையாடல்

உங்களுக்கு என்ன தேவை என்று சொல்லுங்கள்:

```
> Acme Tech-க்கு founders' agreement draft செய் — 3 co-founders, 4-year vesting, 1-year cliff
> Karnataka-வில் registered company-இன் SHA-க்கு stamp duty எவ்வளவு?
> ₹50L angel round-க்கு iSAFE note வேண்டும்
> 10th employee hire செய்துள்ளோம் — POSH compliance தேவையா check செய்
> இந்த வாரம் Indian SaaS-இல் யார் funding raise செய்தார்கள்?
```

Receptionist intent classify செய்து சரியான drafting plugin அல்லது specialist agent-இடம் route செய்கிறது.

ஒவ்வொரு plugin-ம் 6-agent drafting pipeline (Reader → Format → Drafter → Verifier → Refiner → Overseer) இயக்கி `.docx` file-ஐ உங்கள் working directory-இல் தருகிறது.

### Step 5 — ஒவ்வொரு session முடிவிலும் `/retrospective` இயக்கவும்

```
/retrospective
```

உங்கள் company facts, cap table state, open matters, decisions local palace-இல் save செய்கிறது. அடுத்த session-இல் `/wake` அனைத்தையும் load செய்கிறது. Firm கற்றுக்கொள்கிறது.

## 🛡️ Privacy-இன் நேர்மையான பக்கம்

**Tool தானே:** உங்கள் company data, cap table, drafts, notes — அனைத்தும் laptop-இல். Cloud-இல் ஒருபோதும் upload இல்லை. அனைத்தும் `~/.ai-startup-firm-india/palace/`-இல். Source code MIT licensed — ஒவ்வொரு வரியும் படியுங்கள், audit செய்யுங்கள், modify செய்யுங்கள். Drafting plugins `.docx` files local disk-இல் உருவாக்குகின்றன.

**AI model (Claude Code):** Claude Code Anthropic-இன் API-உடன் connect ஆகிறது. உங்கள் queries அவர்களின் servers-க்குச் செல்கின்றன. Anthropic-இன் enterprise privacy defaults strong — prompts மற்றும் outputs default-ஆக model training-இல் use ஆகாது. மேலும்: https://docs.claude.codes

**Compliance பொறுப்பு:** Compliance agents உங்கள் stage-க்கு என்ன பொருந்தும் என்று கூறுகின்றன. அவை உங்கள் company secretary, CA, அல்லது legal counsel-க்கு substitute அல்ல. RoC filing, stamp duty payment, POSH IC constitution, மற்றும் அனைத்து statutory obligations-இன் பொறுப்பும் உங்களுடையது.
