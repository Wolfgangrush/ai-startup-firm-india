## ✨ ഇത് എന്താണ്, ഒരു paragraph-ൽ

ഒരു സൗജന്യ open-source shell — Claude Code-നെ നിങ്ങളുടെ startup-ന്റെ legal co-pilot ആക്കി മാറ്റുന്നു. ഇത് founders' agreements, iSAFE notes, SHAs, employment contracts, POSH policies എന്നിവ draft ചെയ്യുന്നു. നിങ്ങളുടെ ചോദ്യങ്ങളെ 10 startup intents വഴി 8 specialist agents-ലേക്ക് route ചെയ്യുന്നു, അവർ 5 Wolfgang Rush drafting plugins പ്രവർത്തിപ്പിക്കുന്നു. ഇത് നിങ്ങളുടെ കമ്പനി ഓർമ്മിക്കുന്നു — founders, cap table, open matters, compliance deadlines — session-ൽ നിന്ന് session-ലേക്ക്. നിങ്ങളുടെ സ്വന്തം machine-ൽ Claude Code-നുള്ളിൽ പ്രവർത്തിക്കുന്നു. Cloud upload ഇല്ല. Subscription ഇല്ല. ഉണ്ടാകുന്ന document നിങ്ങളുടേതാണ്.

## 💛 ഒരു Indian founder എന്തുകൊണ്ട് ശ്രദ്ധിക്കണം

വലിയ law firms ഒരു standard SHA-ക്ക് ₹50,000–₹2,00,000 വാങ്ങുന്നു. Bengaluru-വിലോ Mumbai-യിലോ ഒരു നല്ല startup lawyer മണിക്കൂറിന് ₹5,000–₹15,000 bill ചെയ്യുന്നു. നിങ്ങൾ pre-revenue ആണ്. Bootstrapping ചെയ്യുകയാണ്. പ്രവർത്തിക്കുന്ന documents വേണം — runway തീർക്കുന്ന relationship അല്ല.

- **Drafting-ൽ ലക്ഷങ്ങൾ ലാഭിക്കുക.** Founders' Agreement, iSAFE, SHA, Employment Contracts — templated, jurisdiction-aware, നിങ്ങളുടെ counsel-ന്റെ റിവ്യൂവിന് തയ്യാർ.
- **Default ആയി compliant ആയിരിക്കുക.** Companies Act 2013, FEMA NDI Rules, DPDP Act 2023, POSH Act 2013 — compliance agent നിങ്ങളുടെ stage-ൽ എന്താണ് ബാധകമെന്ന് പറയുന്നു.
- **ഒരിക്കലും context നഷ്ടപ്പെടുത്തരുത്.** Firm നിങ്ങളുടെ cap table, incorporation state, open drafts — sessions-നിടയിൽ, ആഴ്ചകൾക്ക് ശേഷവും ഓർമ്മിക്കുന്നു.
- **വേഗത്തിൽ മുന്നോട്ട്.** Plain English (അല്ലെങ്കിൽ Hindi, Marathi, Tamil...)-ൽ എന്താണ് വേണ്ടതെന്ന് പറയൂ — firm ശരിയായ specialist-ലേക്ക് route ചെയ്യുന്നു.
- **എല്ലാം നിങ്ങളുടേത്.** MIT licensed. നിങ്ങളുടെ documents. നിങ്ങളുടെ data. നിങ്ങളുടെ laptop.

## 🧠 ഉള്ളിൽ എന്താണ് — 8 specialists നിങ്ങളുടെ terminal-ൽ

8 startup lawyers പോലെ ചിന്തിക്കുക, ഓരോരുത്തർക്കും സ്വന്തം lane:

1. **Receptionist (തലച്ചോറ്)** — നിങ്ങൾ പറയുന്നത് കേൾക്കുന്നു, 10 startup flows-ൽ intent classify ചെയ്യുന്നു, ശരിയായ specialist-ലേക്ക് route ചെയ്യുന്നു.
2. **Matter Manager** — നിങ്ങളുടെ entity details, founders, cap table, open incorporations, pending drafts സൂക്ഷിക്കുന്നു. Sessions-നിടയിൽ stateful.
3. **Companies Act Compliance Officer** — Companies Act 2013: MoA, AoA, DIN, DSC, Section 10A, statutory registers, annual filings. നിങ്ങളുടെ stage-ൽ എന്താണ് ബാധകമെന്ന് അറിയാം.
4. **FEMA Rules Specialist** — foreign investment: NDI Rules, FC-GPR, FMV computation, pricing guidelines. RBI approval വേണ്ടിവരുമ്പോൾ flag ചെയ്യുന്നു.
5. **DPDP Overlay** — Data Protection Act 2023: consent notices, data fiduciary obligations, breach reporting thresholds.
6. **Stamp Duty Router** — 4 സംസ്ഥാനങ്ങളിൽ (MH, KA, DL, TS) stamp duty compute ചെയ്യുന്നു, എവിടെ എത്ര അടക്കണമെന്ന് പറയുന്നു.
7. **POSH Compliance Officer** — workplace harassment prevention: policy drafting, IC constitution, annual filings. Non-compliant ആകുന്നതിന് മുമ്പ് പിടിക്കുന്നു.
8. **Funding & News Watch** — ആര് funding raise ചെയ്തു, VC deal flow, Startup India scheme updates, YC batches, accelerator deadlines.

നിങ്ങൾ Receptionist-മായി മാത്രമേ സംസാരിക്കൂ. Receptionist ബാക്കിയുള്ളവരുമായി സംസാരിക്കുന്നു.

## 🚀 നിങ്ങളുടെ ആദ്യ 30 മിനിറ്റ് (quick start)

### Prerequisites
- Claude Code installed
- Node.js 18+, Python 3.10+
- Git

### Step 1 — Firm clone (2 മിനിറ്റ്)

```bash
git clone https://github.com/Wolfgangrush/ai-startup-firm-india.git ~/Desktop/AIO/ai-startup-firm/india/
```

### Step 2 — 5 Wolfgang Rush drafting plugins install (5 മിനിറ്റ്)

```bash
git clone https://github.com/Wolfgangrush/wolfgang-rush-submissions.git ~/Desktop/AIO/wolfgang-rush-submissions/
```

5 startup drafting plugins `~/Desktop/AIO/wolfgang-rush-submissions/IN-startup/`-ൽ:

| # | Plugin | എന്താണ് draft ചെയ്യുന്നത് |
|---|---|---|
| 1 | `startup-india-founders-agreement-drafting` | Founders' Agreement, Co-founder Exit + Vesting |
| 2 | `startup-india-incorporation-drafting` | MoA + AoA, DSC + DIN, Section 10A declaration |
| 3 | `startup-india-isafe-drafting` | iSAFE INR, iSAFE Foreign (FEMA overlay) |
| 4 | `startup-india-sha-drafting` | SHA Standard, SHA Investment (FEMA overlay) |
| 5 | `startup-india-employment-drafting` | Employment Contract, Offer Letter, POSH + HR Policies |

### Step 3 — Session start (30 സെക്കൻഡ്)

```bash
cd ~/Desktop/AIO/ai-startup-firm/india/
claude
```

Claude Code prompt-ൽ:

```
/wake
```

Saved state load ചെയ്യുന്നു. ആദ്യ തവണയാണെങ്കിൽ fresh firm initialize ചെയ്യുന്നു. Startup dashboard കാണും — company snapshot, open matters, compliance deadlines, funding news.

### Step 4 — ആദ്യ സംഭാഷണം

എന്താണ് വേണ്ടതെന്ന് പറയൂ:

```
> Acme Tech-ന് founders' agreement draft ചെയ്യൂ — 3 co-founders, 4-year vesting, 1-year cliff
> Karnataka-യിൽ registered കമ്പനിയുടെ SHA-ക്ക് stamp duty എത്ര?
> ₹50L angel round-ന് iSAFE note വേണം
> 10th employee hire ചെയ്തു — POSH compliance വേണോ എന്ന് check ചെയ്യൂ
> ഈ ആഴ്ച Indian SaaS-ൽ ആര് funding raise ചെയ്തു?
```

Receptionist intent classify ചെയ്ത് ശരിയായ drafting plugin-ലേക്കോ specialist agent-ലേക്കോ route ചെയ്യുന്നു.

ഓരോ plugin-ഉം 6-agent drafting pipeline (Reader → Format → Drafter → Verifier → Refiner → Overseer) പ്രവർത്തിപ്പിച്ച് `.docx` file നിങ്ങളുടെ working directory-യിൽ നൽകുന്നു.

### Step 5 — ഓരോ session അവസാനവും `/retrospective` പ്രവർത്തിപ്പിക്കുക

```
/retrospective
```

നിങ്ങളുടെ company facts, cap table state, open matters, decisions local palace-ൽ save ചെയ്യുന്നു. അടുത്ത session-ൽ `/wake` എല്ലാം load ചെയ്യുന്നു. Firm പഠിക്കുന്നു.

## 🛡️ Privacy-യുടെ സത്യസന്ധമായ ഭാഗം

**Tool സ്വയം:** നിങ്ങളുടെ company data, cap table, drafts, notes — എല്ലാം laptop-ൽ. Cloud-ൽ ഒരിക്കലും upload ഇല്ല. എല്ലാം `~/.ai-startup-firm-india/palace/`-ൽ. Source code MIT licensed — ഓരോ വരിയും വായിക്കുക, audit ചെയ്യുക, modify ചെയ്യുക. Drafting plugins `.docx` files local disk-ൽ നിർമ്മിക്കുന്നു.

**AI model (Claude Code):** Claude Code Anthropic-ന്റെ API-യുമായി connect ചെയ്യുന്നു. നിങ്ങളുടെ queries അവരുടെ servers-ലേക്ക് പോകുന്നു. Anthropic-ന്റെ enterprise privacy defaults strong — prompts-ഉം outputs-ഉം default ആയി model training-ന് use ചെയ്യുന്നില്ല. കൂടുതൽ: https://docs.claude.codes

**Compliance ഉത്തരവാദിത്തം:** Compliance agents നിങ്ങളുടെ stage-ൽ എന്താണ് ബാധകമെന്ന് പറയുന്നു. അവ നിങ്ങളുടെ company secretary, CA, അല്ലെങ്കിൽ legal counsel-ന്റെ substitute അല്ല. RoC filing, stamp duty payment, POSH IC constitution, എല്ലാ statutory obligations-ന്റേയും ഉത്തരവാദിത്തം നിങ്ങളുടേതാണ്.
