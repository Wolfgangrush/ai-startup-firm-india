## ✨ ଏହା କଣ, ଗୋଟିଏ paragraph ରେ

ଗୋଟିଏ ମାଗଣା open-source shell ଯାହା Claude Code କୁ ଆପଣଙ୍କ startup ର legal co-pilot ରେ ପରିଣତ କରେ। ଏହା founders' agreements, iSAFE notes, SHAs, employment contracts, ଏବଂ POSH policies draft କରେ। ଆପଣଙ୍କ ପ୍ରଶ୍ନଗୁଡ଼ିକୁ 10 startup intents ମାଧ୍ୟମରେ 8 specialist agents କୁ route କରେ, ଯେଉଁମାନେ 5 Wolfgang Rush drafting plugins ଚଲାନ୍ତି। ଏହା ଆପଣଙ୍କ କମ୍ପାନୀକୁ ମନେ ରଖେ — founders, cap table, open matters, compliance deadlines — session ରୁ session। ଆପଣଙ୍କ ନିଜ machine ରେ Claude Code ଭିତରେ ଚାଲେ। Cloud upload ନାହିଁ। Subscription ନାହିଁ। ଯେଉଁ document ତିଆରି ହୁଏ, ତାହା ଆପଣଙ୍କର।

## 💛 ଜଣେ Indian founder କାହିଁକି ଧ୍ୟାନ ଦେବା କଥା

ବଡ଼ law firms ଗୋଟିଏ standard SHA ପାଇଁ ₹50,000–₹2,00,000 ନିଅନ୍ତି। Bengaluru କିମ୍ବା Mumbai ରେ ଜଣେ ଭଲ startup lawyer ଘଣ୍ଟାକୁ ₹5,000–₹15,000 bill କରନ୍ତି। ଆପଣ pre-revenue ଅଛନ୍ତି। Bootstrapping କରୁଛନ୍ତି। ଆପଣଙ୍କୁ କାମ କରୁଥିବା documents ଦରକାର — runway ଶେଷ କରିଦେଉଥିବା relationship ନୁହେଁ।

- **Drafting ରେ ଲକ୍ଷ ଟଙ୍କା ସଞ୍ଚୟ କରନ୍ତୁ।** Founders' Agreement, iSAFE, SHA, Employment Contracts — templated, jurisdiction-aware, ଆପଣଙ୍କ counsel ଙ୍କ ସମୀକ୍ଷା ପାଇଁ ପ୍ରସ୍ତୁତ।
- **Default ଭାବେ compliant ରୁହନ୍ତୁ।** Companies Act 2013, FEMA NDI Rules, DPDP Act 2023, POSH Act 2013 — compliance agent କହେ ଆପଣଙ୍କ stage ରେ କଣ ଲାଗୁ ହୁଏ।
- **କେବେ context ହରାନ୍ତୁ ନାହିଁ।** Firm ଆପଣଙ୍କ cap table, incorporation state, open drafts — sessions ମଝିରେ, ସପ୍ତାହ ପରେ ମଧ୍ୟ ମନେ ରଖେ।
- **ତୀବ୍ର ଗତିରେ ଆଗକୁ ବଢ଼ନ୍ତୁ।** Plain English (କିମ୍ବା Hindi, Marathi, Tamil...) ରେ କଣ ଦରକାର କୁହନ୍ତୁ — firm ଠିକ୍ specialist କୁ route କରେ।
- **ସବୁ କିଛି ଆପଣଙ୍କର।** MIT licensed. ଆପଣଙ୍କ documents. ଆପଣଙ୍କ data. ଆପଣଙ୍କ laptop.

## 🧠 ଭିତରେ କଣ ଅଛି — 8 specialists ଆପଣଙ୍କ terminal ରେ

8 startup lawyers ଭଳି ଭାବନ୍ତୁ, ପ୍ରତ୍ୟେକର ନିଜ lane:

1. **Receptionist (ମସ୍ତିଷ୍କ)** — ଆପଣ ଯାହା କୁହନ୍ତି ଶୁଣେ, 10 startup flows ରେ intent classify କରେ, ଠିକ୍ specialist କୁ route କରେ।
2. **Matter Manager** — ଆପଣଙ୍କ entity details, founders, cap table, open incorporations, pending drafts ରଖେ। Sessions ମଝିରେ stateful।
3. **Companies Act Compliance Officer** — Companies Act 2013: MoA, AoA, DIN, DSC, Section 10A, statutory registers, annual filings। ଆପଣଙ୍କ stage ରେ କଣ ଲାଗୁ ହୁଏ ଜାଣେ।
4. **FEMA Rules Specialist** — foreign investment: NDI Rules, FC-GPR, FMV computation, pricing guidelines। RBI approval ଦରକାର ପଡ଼ିଲେ flag କରେ।
5. **DPDP Overlay** — Data Protection Act 2023: consent notices, data fiduciary obligations, breach reporting thresholds।
6. **Stamp Duty Router** — 4 ଟି ରାଜ୍ୟ (MH, KA, DL, TS) ରେ stamp duty compute କରେ, କେଉଁଠି କେତେ ଭରିବେ କହେ।
7. **POSH Compliance Officer** — workplace harassment prevention: policy drafting, IC constitution, annual filings। Non-compliant ହେବା ପୂର୍ବରୁ ଧରେ।
8. **Funding & News Watch** — କିଏ funding raise କଲା, VC deal flow, Startup India scheme updates, YC batches, accelerator deadlines।

ଆପଣ କେବଳ Receptionist ସହ କଥା କୁହନ୍ତି। Receptionist ବାକିମାନଙ୍କ ସହ କଥା କହେ।

## 🚀 ଆପଣଙ୍କ ପ୍ରଥମ 30 ମିନିଟ୍ (quick start)

### Prerequisites
- Claude Code installed
- Node.js 18+ ଏବଂ Python 3.10+
- Git

### Step 1 — Firm clone (2 ମିନିଟ୍)

```bash
git clone https://github.com/Wolfgangrush/ai-startup-firm-india.git ~/Desktop/AIO/ai-startup-firm/india/
```

### Step 2 — 5 Wolfgang Rush drafting plugins install (5 ମିନିଟ୍)

```bash
git clone https://github.com/Wolfgangrush/wolfgang-rush-submissions.git ~/Desktop/AIO/wolfgang-rush-submissions/
```

5 startup drafting plugins `~/Desktop/AIO/wolfgang-rush-submissions/IN-startup/` ରେ:

| # | Plugin | କଣ draft କରେ |
|---|---|---|
| 1 | `startup-india-founders-agreement-drafting` | Founders' Agreement, Co-founder Exit + Vesting |
| 2 | `startup-india-incorporation-drafting` | MoA + AoA, DSC + DIN, Section 10A declaration |
| 3 | `startup-india-isafe-drafting` | iSAFE INR, iSAFE Foreign (FEMA overlay) |
| 4 | `startup-india-sha-drafting` | SHA Standard, SHA Investment (FEMA overlay) |
| 5 | `startup-india-employment-drafting` | Employment Contract, Offer Letter, POSH + HR Policies |

### Step 3 — Session start (30 ସେକେଣ୍ଡ)

```bash
cd ~/Desktop/AIO/ai-startup-firm/india/
claude
```

Claude Code prompt ରେ:

```
/wake
```

Saved state load କରେ। ପ୍ରଥମ ଥର ହୋଇଥିଲେ fresh firm initialize କରେ। Startup dashboard ଦେଖିବେ — company snapshot, open matters, compliance deadlines, funding news।

### Step 4 — ପ୍ରଥମ କଥୋପକଥନ

କଣ ଦରକାର କୁହନ୍ତୁ:

```
> Acme Tech ପାଇଁ founders' agreement draft କର — 3 co-founders, 4-year vesting, 1-year cliff
> Karnataka ରେ registered କମ୍ପାନୀର SHA ଉପରେ stamp duty କେତେ?
> ₹50L angel round ପାଇଁ iSAFE note ଦରକାର
> Check କର ଯେ 10th employee hire କଲା ପରେ POSH compliance ଦରକାର କି?
> ଏହି ସପ୍ତାହ Indian SaaS ରେ କିଏ funding raise କଲା?
```

Receptionist intent classify କରି ଠିକ୍ drafting plugin କିମ୍ବା specialist agent କୁ route କରେ।

ପ୍ରତ୍ୟେକ plugin 6-agent drafting pipeline (Reader → Format → Drafter → Verifier → Refiner → Overseer) ଚଲାଇ `.docx` file ଆପଣଙ୍କ working directory ରେ ଦିଏ।

### Step 5 — ପ୍ରତ୍ୟେକ session ଶେଷରେ `/retrospective` ଚଲାନ୍ତୁ

```
/retrospective
```

ଆପଣଙ୍କ company facts, cap table state, open matters, decisions local palace ରେ save କରେ। ପର session ରେ `/wake` ସବୁ load କରେ। Firm ଶିଖେ।

## 🛡️ Privacy ର ସତ ପକ୍ଷ

**Tool ନିଜେ:** ଆପଣଙ୍କ company data, cap table, drafts, notes — ସବୁ laptop ରେ। Cloud ରେ କେବେ upload ନୁହେଁ। ସବୁ `~/.ai-startup-firm-india/palace/` ରେ। Source code MIT licensed — ପ୍ରତ୍ୟେକ ଧାଡ଼ି ପଢ଼ନ୍ତୁ, audit କରନ୍ତୁ, modify କରନ୍ତୁ। Drafting plugins `.docx` files local disk ରେ ତିଆରି କରନ୍ତି।

**AI model (Claude Code):** Claude Code Anthropic ର API ସହିତ connect ହୁଏ। ଆପଣଙ୍କ queries ସେମାନଙ୍କ servers କୁ ଯାଏ। Anthropic ର enterprise privacy defaults strong — prompts ଏବଂ outputs default ଭାବେ model training ରେ use ହୁଏ ନାହିଁ। ଅଧିକ: https://docs.claude.codes

**Compliance ଦାୟିତ୍ୱ:** Compliance agents ଆପଣଙ୍କ stage ରେ କଣ ଲାଗୁ ହୁଏ କହନ୍ତି। ସେମାନେ ଆପଣଙ୍କ company secretary, CA, କିମ୍ବା legal counsel ଙ୍କ substitute ନୁହଁନ୍ତି। RoC filing, stamp duty payment, POSH IC constitution, ଏବଂ ସମସ୍ତ statutory obligations ର ଦାୟିତ୍ୱ ଆପଣଙ୍କର।
