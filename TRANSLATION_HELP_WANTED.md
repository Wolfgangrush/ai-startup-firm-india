## How to contribute (5-minute version)

### Easiest — open an Issue

1. Go to https://github.com/Wolfgangrush/ai-startup-firm-india/issues
2. Click "New Issue"
3. Title: `[translation-fix] <language> — <brief description>`
4. Body: paste the line(s) that need fixing + suggest a better version
5. Submit. We'll patch it.

### Better — open a Pull Request

1. Fork the repo on GitHub
2. Edit the file directly in GitHub (pencil icon · works in browser, no Git needed)
3. Replace the lines you want to fix
4. Add a commit message: `fix(<language>): polish <section name>`
5. Open Pull Request → we review and merge

### Best — translate a section we missed

If you want to add a section that doesn't exist in your language guide yet — go for it. Open a PR. We'll merge if the translation matches the structural meaning of the English original.

## What we need most

Native speakers' help on these specific surfaces, in priority order:

### Priority 1 — Startup legal terminology
The English originals use Indian startup legal terms like:
- "Founders' Agreement" · "iSAFE Note" · "Shareholders' Agreement (SHA)"
- "Cap Table" · "Fully Diluted" · "Option Pool" · "ESOP"
- "Companies Act 2013" · "MoA / AoA" · "DIN / DSC" · "Section 10A"
- "FEMA NDI Rules" · "FC-GPR" · "FMV"
- "DPDP Act 2023" · "POSH Act 2013" · "IC Constitution"
- "Stamp Duty" · "RoC Filing" · "Startup India"
- "Term Sheet" · "Angel Round" · "Vesting" · "Cliff"

In each Indian language, startup founders use a mix of:
- Standard legal/official equivalents
- English loanwords (most common in practice)
- Colloquial forms

We need a native speaker who's been through incorporation or fundraising to pick the form an Indian founder actually uses in conversation. Right now the AI-assisted versions sometimes pick the wrong register.

### Priority 2 — Idiom and tone
The English original has a voice: direct, founder-to-founder, "you're building, here's your toolkit." That voice should land identically in each language. Where the AI translation feels stiff, textbook-y, or overly formal, please reword for how founders actually talk.

### Priority 3 — Honorifics
Different regions use different forms of address. We picked:
- Hindi: "फाउंडर जी"
- Marathi: "संस्थापक"
- Bengali: "প্রতিষ্ঠাতা"
- Tamil: "நிறுவனர்"
- Telugu: "వ్యవస్థాపకులు"
- Gujarati: "સ્થાપક"
- Kannada: "ಸಂಸ್ಥಾಪಕರೇ"
- Malayalam: "സ്ഥാപകരേ"
- Punjabi: "ਸੰਸਥਾਪਕ ਜੀ"
- Odia: "ପ୍ରତିଷ୍ଠାତା"

If your region uses different — let us know.

### Priority 4 — Script choice
For Punjabi: we used Gurmukhi (the dominant script in Indian Punjab). For founders in Pakistan or other regions using Shahmukhi — we'd welcome a parallel translation if there's demand.

For Urdu: not yet included. If there's demand from founders who prefer Urdu — please open an issue and we'll add it.

## What we WON'T accept

We cannot accept translations that:
- Frame the tool as a law firm or legal service ("Hire us for...")
- Make claims about legal outcomes ("This will get you funded")
- Solicit work for any specific lawyer, CA, or CS
- Replace the disclaimer with weaker language

Translation = preserve meaning, not amplify. If the original says "review by qualified counsel before execution," keep that warning intact.

## Why this matters

Indian founders are building from Kanyakumari to Kashmir, in ~22 official languages + 100+ unofficial ones. A founder in Surat who reads Gujarati comfortably but English with effort should get the same warm onboarding as one in Bengaluru who reads English natively.

The AI-assisted translations get us 70% of the way there. Native speakers close the last 30% — the part that makes the guide feel like it was written FOR them, not AT them.

Your one PR could be the difference between a founder giving up on incorporation paperwork at midnight versus actually filing.
