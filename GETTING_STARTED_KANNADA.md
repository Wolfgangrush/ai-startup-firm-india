## ✨ ಇದು ಏನು, ಒಂದು paragraph ನಲ್ಲಿ

ಒಂದು ಉಚಿತ open-source shell — Claude Code ಅನ್ನು ನಿಮ್ಮ startup ನ legal co-pilot ಆಗಿ ಪರಿವರ್ತಿಸುತ್ತದೆ. ಇದು founders' agreements, iSAFE notes, SHAs, employment contracts, ಮತ್ತು POSH policies draft ಮಾಡುತ್ತದೆ. ನಿಮ್ಮ ಪ್ರಶ್ನೆಗಳನ್ನು 10 startup intents ಮೂಲಕ 8 specialist agents ಗೆ route ಮಾಡುತ್ತದೆ, ಅವು 5 Wolfgang Rush drafting plugins ಚಲಾಯಿಸುತ್ತವೆ. ಇದು ನಿಮ್ಮ ಕಂಪನಿಯನ್ನು ನೆನಪಿಡುತ್ತದೆ — founders, cap table, open matters, compliance deadlines — session ನಿಂದ session. ನಿಮ್ಮ ಸ್ವಂತ machine ನಲ್ಲಿ Claude Code ಒಳಗೆ ಚಲಿಸುತ್ತದೆ. Cloud upload ಇಲ್ಲ. Subscription ಇಲ್ಲ. ತಯಾರಾಗುವ document ನಿಮ್ಮದು.

## 💛 ಒಬ್ಬ Indian founder ಏಕೆ ತಲೆಕೆಡಿಸಿಕೊಳ್ಳಬೇಕು

ದೊಡ್ಡ law firms ಒಂದು standard SHA ಗೆ ₹50,000–₹2,00,000 ತೆಗೆದುಕೊಳ್ಳುತ್ತವೆ. Bengaluru ಅಥವಾ Mumbai ನಲ್ಲಿ ಒಬ್ಬ ಒಳ್ಳೆಯ startup lawyer ಗಂಟೆಗೆ ₹5,000–₹15,000 bill ಮಾಡುತ್ತಾರೆ. ನೀವು pre-revenue. Bootstrapping ಮಾಡುತ್ತಿದ್ದೀರಿ. ನಿಮಗೆ ಕೆಲಸ ಮಾಡುವ documents ಬೇಕು — runway ಮುಗಿಸುವ relationship ಅಲ್ಲ.

- **Drafting ನಲ್ಲಿ ಲಕ್ಷ ಉಳಿಸಿ.** Founders' Agreement, iSAFE, SHA, Employment Contracts — templated, jurisdiction-aware, ನಿಮ್ಮ counsel ವಿಮರ್ಶೆಗೆ ಸಿದ್ಧ.
- **Default ಆಗಿ compliant ಆಗಿರಿ.** Companies Act 2013, FEMA NDI Rules, DPDP Act 2023, POSH Act 2013 — compliance agent ನಿಮ್ಮ stage ಗೆ ಏನು ಅನ್ವಯಿಸುತ್ತದೆ ಎಂದು ಹೇಳುತ್ತದೆ.
- **Context ಎಂದಿಗೂ ಕಳೆದುಕೊಳ್ಳಬೇಡಿ.** Firm ನಿಮ್ಮ cap table, incorporation state, open drafts — sessions ನಡುವೆ, ವಾರಗಳ ನಂತರವೂ ನೆನಪಿಡುತ್ತದೆ.
- **ವೇಗವಾಗಿ ಮುನ್ನಡೆಯಿರಿ.** Plain English (ಅಥವಾ Hindi, Marathi, Tamil...) ನಲ್ಲಿ ಏನು ಬೇಕು ಎಂದು ಹೇಳಿ — firm ಸರಿಯಾದ specialist ಗೆ route ಮಾಡುತ್ತದೆ.
- **ಎಲ್ಲವೂ ನಿಮ್ಮದು.** MIT licensed. ನಿಮ್ಮ documents. ನಿಮ್ಮ data. ನಿಮ್ಮ laptop.

## 🧠 ಒಳಗೆ ಏನಿದೆ — 8 specialists ನಿಮ್ಮ terminal ನಲ್ಲಿ

8 startup lawyers ರಂತೆ ಯೋಚಿಸಿ, ಪ್ರತಿಯೊಬ್ಬರಿಗೂ ಸ್ವಂತ lane:

1. **Receptionist (ಮೆದುಳು)** — ನೀವು ಹೇಳುವುದನ್ನು ಕೇಳುತ್ತದೆ, 10 startup flows ನಲ್ಲಿ intent classify ಮಾಡುತ್ತದೆ, ಸರಿಯಾದ specialist ಗೆ route ಮಾಡುತ್ತದೆ.
2. **Matter Manager** — ನಿಮ್ಮ entity details, founders, cap table, open incorporations, pending drafts ಇಡುತ್ತದೆ. Sessions ನಡುವೆ stateful.
3. **Companies Act Compliance Officer** — Companies Act 2013: MoA, AoA, DIN, DSC, Section 10A, statutory registers, annual filings. ನಿಮ್ಮ stage ಗೆ ಏನು ಅನ್ವಯಿಸುತ್ತದೆ ಎಂದು ತಿಳಿದಿದೆ.
4. **FEMA Rules Specialist** — foreign investment: NDI Rules, FC-GPR, FMV computation, pricing guidelines. RBI approval ಬೇಕಾದಾಗ flag ಮಾಡುತ್ತದೆ.
5. **DPDP Overlay** — Data Protection Act 2023: consent notices, data fiduciary obligations, breach reporting thresholds.
6. **Stamp Duty Router** — 4 ರಾಜ್ಯಗಳಲ್ಲಿ (MH, KA, DL, TS) stamp duty compute ಮಾಡುತ್ತದೆ, ಎಲ್ಲಿ ಎಷ್ಟು ಪಾವತಿಸಬೇಕು ಎಂದು ಹೇಳುತ್ತದೆ.
7. **POSH Compliance Officer** — workplace harassment prevention: policy drafting, IC constitution, annual filings. Non-compliant ಆಗುವ ಮೊದಲು ಹಿಡಿಯುತ್ತದೆ.
8. **Funding & News Watch** — ಯಾರು funding raise ಮಾಡಿದರು, VC deal flow, Startup India scheme updates, YC batches, accelerator deadlines.

ನೀವು Receptionist ಜೊತೆ ಮಾತ್ರ ಮಾತನಾಡುತ್ತೀರಿ. Receptionist ಉಳಿದವರ ಜೊತೆ ಮಾತನಾಡುತ್ತದೆ.

## 🚀 ನಿಮ್ಮ ಮೊದಲ 30 ನಿಮಿಷಗಳು (quick start)

### Prerequisites
- Claude Code installed
- Node.js 18+ ಮತ್ತು Python 3.10+
- Git

### Step 1 — Firm clone (2 ನಿಮಿಷಗಳು)

```bash
git clone https://github.com/Wolfgangrush/ai-startup-firm-india.git ~/Desktop/AIO/ai-startup-firm/india/
```

### Step 2 — 5 Wolfgang Rush drafting plugins install (5 ನಿಮಿಷಗಳು)

```bash
git clone https://github.com/Wolfgangrush/wolfgang-rush-submissions.git ~/Desktop/AIO/wolfgang-rush-submissions/
```

5 startup drafting plugins `~/Desktop/AIO/wolfgang-rush-submissions/IN-startup/` ನಲ್ಲಿ:

| # | Plugin | ಏನು draft ಮಾಡುತ್ತದೆ |
|---|---|---|
| 1 | `startup-india-founders-agreement-drafting` | Founders' Agreement, Co-founder Exit + Vesting |
| 2 | `startup-india-incorporation-drafting` | MoA + AoA, DSC + DIN, Section 10A declaration |
| 3 | `startup-india-isafe-drafting` | iSAFE INR, iSAFE Foreign (FEMA overlay) |
| 4 | `startup-india-sha-drafting` | SHA Standard, SHA Investment (FEMA overlay) |
| 5 | `startup-india-employment-drafting` | Employment Contract, Offer Letter, POSH + HR Policies |

### Step 3 — Session start (30 ಸೆಕೆಂಡುಗಳು)

```bash
cd ~/Desktop/AIO/ai-startup-firm/india/
claude
```

Claude Code prompt ನಲ್ಲಿ:

```
/wake
```

Saved state load ಮಾಡುತ್ತದೆ. ಮೊದಲ ಬಾರಿಗೆ ಆದರೆ fresh firm initialize ಮಾಡುತ್ತದೆ. Startup dashboard ನೋಡುತ್ತೀರಿ — company snapshot, open matters, compliance deadlines, funding news.

### Step 4 — ಮೊದಲ ಸಂಭಾಷಣೆ

ನಿಮಗೆ ಏನು ಬೇಕು ಎಂದು ಹೇಳಿ:

```
> Acme Tech ಗೆ founders' agreement draft ಮಾಡು — 3 co-founders, 4-year vesting, 1-year cliff
> Karnataka ದಲ್ಲಿ registered ಕಂಪನಿಯ SHA ಮೇಲೆ stamp duty ಎಷ್ಟು?
> ₹50L angel round ಗೆ iSAFE note ಬೇಕು
> 10th employee hire ಮಾಡಿದ್ದೇವೆ — POSH compliance ಬೇಕೇ ಎಂದು check ಮಾಡು
> ಈ ವಾರ Indian SaaS ನಲ್ಲಿ ಯಾರು funding raise ಮಾಡಿದರು?
```

Receptionist intent classify ಮಾಡಿ ಸರಿಯಾದ drafting plugin ಅಥವಾ specialist agent ಗೆ route ಮಾಡುತ್ತದೆ.

ಪ್ರತಿ plugin 6-agent drafting pipeline (Reader → Format → Drafter → Verifier → Refiner → Overseer) ಚಲಾಯಿಸಿ `.docx` file ನಿಮ್ಮ working directory ನಲ್ಲಿ ನೀಡುತ್ತದೆ.

### Step 5 — ಪ್ರತಿ session ಕೊನೆಯಲ್ಲಿ `/retrospective` ಚಲಾಯಿಸಿ

```
/retrospective
```

ನಿಮ್ಮ company facts, cap table state, open matters, decisions local palace ನಲ್ಲಿ save ಮಾಡುತ್ತದೆ. ಮುಂದಿನ session ನಲ್ಲಿ `/wake` ಎಲ್ಲವನ್ನೂ load ಮಾಡುತ್ತದೆ. Firm ಕಲಿಯುತ್ತದೆ.

## 🛡️ Privacy ಯ ಪ್ರಾಮಾಣಿಕ ಪಕ್ಷ

**Tool ಸ್ವತಃ:** ನಿಮ್ಮ company data, cap table, drafts, notes — ಎಲ್ಲವೂ laptop ನಲ್ಲಿ. Cloud ನಲ್ಲಿ ಎಂದಿಗೂ upload ಇಲ್ಲ. ಎಲ್ಲವೂ `~/.ai-startup-firm-india/palace/` ನಲ್ಲಿ. Source code MIT licensed — ಪ್ರತಿ ಸಾಲು ಓದಿ, audit ಮಾಡಿ, modify ಮಾಡಿ. Drafting plugins `.docx` files local disk ನಲ್ಲಿ ತಯಾರಿಸುತ್ತವೆ.

**AI model (Claude Code):** Claude Code Anthropic API ಗೆ connect ಆಗುತ್ತದೆ. ನಿಮ್ಮ queries ಅವರ servers ಗೆ ಹೋಗುತ್ತವೆ. Anthropic ನ enterprise privacy defaults strong — prompts ಮತ್ತು outputs default ಆಗಿ model training ನಲ್ಲಿ use ಆಗುವುದಿಲ್ಲ. ಇನ್ನಷ್ಟು: https://docs.claude.codes

**Compliance ಜವಾಬ್ದಾರಿ:** Compliance agents ನಿಮ್ಮ stage ಗೆ ಏನು ಅನ್ವಯಿಸುತ್ತದೆ ಎಂದು ಹೇಳುತ್ತವೆ. ಅವು ನಿಮ್ಮ company secretary, CA, ಅಥವಾ legal counsel ಗೆ substitute ಅಲ್ಲ. RoC filing, stamp duty ಪಾವತಿ, POSH IC constitution, ಮತ್ತು ಎಲ್ಲಾ statutory obligations ನ ಜವಾಬ್ದಾರಿ ನಿಮ್ಮದು.
